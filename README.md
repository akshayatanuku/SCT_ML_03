Task 03: Image Classification using Support Vector Machines (SVM)
This project focuses on Computer Vision and Supervised Learning to classify images into distinct categories, such as "Dogs" vs. "Cats."

Key Highlights:
Algorithm: Implemented a Support Vector Machine (SVM) to find the optimal hyperplane that separates different image classes.

Feature Extraction: Processed raw pixel data by resizing images and flattening them into feature vectors for the model to analyze.

Data Preparation: Utilized the Kaggle Dogs vs. Cats dataset, performing data augmentation and normalization to improve model generalization.

Model Evaluation: Achieved high classification accuracy by tuning hyperparameters like the Kernel (RBF/Linear) and Gamma values.

Goal: To build a robust system capable of automatically identifying and labeling visual content in a digital dataset.
